# Contribution Guidelines

Thank you for considering contributing to Awesome Agent Economy!

## Adding a project

Please ensure your pull request adheres to the following guidelines:

- **Relevant to the agent economy** — The project should involve AI agents participating in economic activities (payments, commerce, identity, reputation, publishing, etc.).
- **Active and accessible** — The project should be actively maintained and publicly accessible.
- **One project per PR** — Submit a separate pull request for each project.
- **Use the following format:** `- [Project Name](https://example.com/) - A concise description that starts with a capital letter and ends with a period.`
- **Add to the appropriate category** — If no category fits, suggest a new one.
- **Check your spelling and grammar.**
- **No duplicates** — Search existing entries before submitting.
- **No purely promotional content** — Projects should provide genuine value to the agent economy ecosystem.

## Updating an existing entry

If a project has changed significantly (new URL, rebranded, deprecated), feel free to submit a PR with the update.

## Quality standards

We reserve the right to reject submissions that:

- Are not relevant to the AI agent economy
- Are inactive, abandoned, or vaporware
- Contain misleading descriptions
- Are duplicates of existing entries

## New categories

If you think a new category is needed, open an issue first to discuss it.
